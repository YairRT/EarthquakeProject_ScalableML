# EarthquakeProject_ScalableML


## src/usgs_client.py file will be responsible for all communication with the USGS API, including building queries and parsing responses
## src/features.py file will contain feature extraction and basic data processing logic
## src/viz.py file will handle visualizations such as maps and plots
## app.py file will act as the main entry point, connecting data retrieval, feature computation, visualization, and the UI
